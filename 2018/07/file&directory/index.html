Python中对文件和目录的常用操作---os模块的基本使用
发布于：2018-07-22 17:56:17 标签：/ python / 访问：

Python中对文件和目录的操作经常用到os模块.下面对其常用功能和实现方法作简单陈述:

  • 获得当前脚本文件的目录路径:

    os.getcwd()

  • 返回指定目录下所包含的所有文件和目录:

    os.listdir() 例: 返回python目录下的文件 –> os.listdir("/python/")

  • 删除一个文件:

    os.remove(filepath)

  • 删除多个目录(空):

    os.remove(r"path")

  • 检验路径是否是一个文件:

    os.path.isfile(filepath)

  • 检验路径是否是一个目录:

    os.path.isdir(filepath)

  • 判断是否为绝对路径:

    os.path.isabs()

  • 检验路径是否存在:

    os.path.exists() 例: 查看python目录下是否存在new目录 –> os.path.exists(r"/python/new")

  • 获取路径名:

    os.path.dirname(filename)

  • 获取文件名:

    os.path.basename(filepath)

  • 重命名文件或目录:

    os.rename(old, new)

  • 创建多级目录:

    os.makedirs(r"/python/new/test")

  • 创建单个目录:

    os.mkdir("test")

  • 获取文件属性:

    os.stat(file)

  • 获取文件大小:

    os.path.getsize(filename)

  • 修改文件权限:

    os.chmod(file)

  • 删除目录:

    os.rmdir("dirpath") (注:只能删除空目录)
